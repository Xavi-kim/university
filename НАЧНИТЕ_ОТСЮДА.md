#### ❌ ЕСЛИ ВХОД НЕ УДАЛСЯ:

##### Вариант A: Пользователь не найден

```
🔍 [AUTH] Попытка входа с email: admin@university.kz
❌ [AUTH] Пользователь не найден: admin@university.kz

❌ [AUTH FAILURE] Ошибка аутентификации:
   Email: admin@university.kz
   Exception: UsernameNotFoundException
   Message: Пользователь с email admin@university.kz не найден
   Причина: Пользователь не существует в базе данных
   Redirect: /auth/login?error=true
```

**РЕШЕНИЕ:**

1. Остановите приложение (Ctrl+C)
2. Удалите данные:
   ```sql
   psql -U postgres -d university_db
   DELETE FROM enrollments;
   DELETE FROM courses;
   DELETE FROM professors;
   DELETE FROM universities;
   DELETE FROM users;
   \q
   ```
3. Перезапустите `start.bat`
4. DataInitializer создаст пользователей заново

---

## 💡 СОВЕТЫ

1. **Всегда смотрите на логи** - они покажут точную причину ошибки
2. **Используйте режим инкогнито** - чтобы исключить проблемы с кэшем
3. **Проверяйте Caps Lock** - пароль чувствителен к регистру
4. **Копируйте пароль** из этого документа - чтобы избежать опечаток

## 📞 ЕСЛИ ВСЁ ЕЩЁ НЕ РАБОТАЕТ

1. Запустите приложение: `start.bat`
2. Попробуйте войти
3. **СКОПИРУЙТЕ ВСЕ ЛОГИ** из консоли (особенно строки с [AUTH])
4. Найдите:
   - Строку `🔍 [AUTH] Попытка входа`
   - Всё что идет после неё до `[AUTH SUCCESS]` или `[AUTH FAILURE]`
5. Отправьте эти логи - по ним можно точно определить проблему
