## ❌ ПРОБЛЕМА: Порт 8080 занят

**Ошибка:**
```
Web server failed to start. Port 8080 was already in use.
```

**Решение:**
```bat
start.bat
```
Файл автоматически освободит порт и запустит приложение!

**Или вручную:**
```powershell
# Найти процесс
netstat -ano | Select-String ":8080"

# Остановить процесс (замените 12360 на ваш PID)
Stop-Process -Id 12360 -Force
```

## 📖 REST API

### Курсы
```
GET    /api/courses              # Все активные курсы
GET    /api/courses/{id}         # Курс по ID
GET    /api/courses/department/{name}  # Курсы по отделению
POST   /api/courses              # Создать курс
PATCH  /api/courses/{id}/status  # Изменить статус
```

### Преподаватели
```
GET    /api/professors           # Все активные преподаватели
GET    /api/professors/{id}      # Преподаватель по ID
```

### Университеты
```
GET    /api/universities         # Все активные университеты
GET    /api/universities/{id}    # Университет по ID
```

## 📝 ДОРАБОТКИ ПО ЛАБОРАТОРНОЙ РАБОТЕ №3

### Сервлеты с doGet и doPost

Реализованы в `MainController`:

**doGet эквиваленты:**
```java
@GetMapping("/api/main")           # Получение данных
@GetMapping("/api/main/special")   # Получение JSON
```

**doPost эквивалент:**
```java
@PostMapping("/api/main/special")  # Отправка данных с параметром name
```

### Дополнительные параметры (Задание)

✅ Добавлены дополнительные параметры:
- `name` - имя студента
- `age` - возраст студента
- `email` - email
- `password` - пароль

✅ Данные выводятся в браузер через:
- HTML страницы (Thymeleaf)
- REST API (JSON)
- API Browser (красивый UI)
