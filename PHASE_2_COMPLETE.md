### 2️⃣ Расширение курсов

#### Новые поля в Course:
- ✅ `credits` - количество кредитов (3, 4, 5)
- ✅ `maxStudents` - максимум студентов
- ✅ `startDate` - дата начала курса
- ✅ `endDate` - дата окончания
- ✅ `category` - категория ("Программирование", "Математика" и т.д.)
- ✅ `tags` - теги через запятую ("Java,Spring,Backend")

- ✅ `level` - уровень (BEGINNER, INTERMEDIATE, ADVANCED)

**Возможности:**

- Визуальные обложки курсов

### 3️⃣ Система оценок

#### Модель Grade создана:
- ✅ `letterGrade` - буквенная оценка (A, B+, B, C, D, F)
- ✅ `numericGrade` - числовая оценка (0-100)
- ✅ `gpaValue` - GPA (0.0 - 4.0)
- ✅ `assignmentName` - название задания
- ✅ `assignmentType` - тип (HOMEWORK, QUIZ, MIDTERM, FINAL, PROJECT)
- ✅ `weight` - вес оценки (30%, 50% и т.д.)
- ✅ `comments` - комментарии преподавателя
- ✅ `gradedAt` - дата выставления
- ✅ `gradedBy` - кто поставил оценку

#### GradeRepository создан:
- ✅ `findByEnrollment()` - все оценки по регистрации
- ✅ `findByStudentId()` - все оценки студента
- ✅ `findByCourseId()` - все оценки по курсу
- ✅ `calculateAverageGPA()` - средний GPA студента
- ✅ `findByAssignmentType()` - оценки по типу задания

#### GradeService создан:
- ✅ CRUD операции с оценками
- ✅ Получение оценок студента/курса
- ✅ Расчёт среднего GPA
- ✅ Фильтрация по типу заданий

**Система оценок:**
```
A   = 4.0 GPA
A-  = 3.7 GPA
B+  = 3.3 GPA
B   = 3.0 GPA
B-  = 2.7 GPA
C+  = 2.3 GPA
C   = 2.0 GPA
C-  = 1.7 GPA
D+  = 1.3 GPA
D   = 1.0 GPA
F   = 0.0 GPA
```

## 🎯 Как использовать

### Просмотр профиля:
1. Войти в систему
2. Перейти на `/profile`
3. Увидеть аватар, информацию, статистику

### Редактирование профиля:
1. На странице профиля нажать "Редактировать профиль"
2. Изменить имя, телефон, био, URL аватара
3. Сохранить

### Смена пароля:
1. На странице профиля нажать "Сменить пароль"
2. Ввести текущий пароль
3. Ввести новый пароль дважды
4. Сохранить

### Просмотр статистики (для студентов):
- Количество курсов
- Средний GPA
- Всего регистраций
